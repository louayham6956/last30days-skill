## ✨ What this app does

last30days-skill helps you research a topic across many sources in one place. It checks recent posts and discussions from Reddit, X, YouTube, Hacker News, Polymarket, and the web. Then it builds a grounded summary that pulls the main points together.

Use it when you want:

- a quick view of what people are saying now
- a recent summary instead of old search results
- a single report from many sources
- a plain answer based on current posts and pages

## 🧭 How to use it

Use simple topic names for the best results.

Examples:

- AI search tools
- Tesla stock discussion
- Reddit views on remote work
- YouTube reactions to a new phone
- Polymarket odds for an event
- Hacker News discussion on privacy

The app searches across recent content, then groups what it finds into a clean summary. It focuses on the last 30 days, so the result stays current.

## 🧾 What you get

A normal result may include:

- a short topic overview
- key points from each source
- patterns or repeated claims
- recent changes in opinion
- links or references to where the information came from

The app aims to keep the output grounded. That means it tries to base the summary on real posts, real pages, and recent data.

## 🧩 Tips for better results

Use these tips to get cleaner output:

- Keep the topic short and clear
- Use one topic at a time
- Try a specific phrase instead of a broad word
- Ask about a recent event, person, or product
- Run the search again if the topic changes fast

Good topic examples:

- “Reddit reaction to GPT-5”
- “Hacker News on local AI”
- “X posts about Bitcoin ETF”
- “YouTube reviews of iPhone battery life”

## 🔐 Privacy and data use

The app works by reading public web content and public social posts. It does not need you to enter private account details for normal use. If a source asks for sign-in, use only accounts you trust and only if the app setup tells you to do so.

## 🧰 If something does not work

If the app does not open:

- check that the download finished
- try extracting the zip file again
- move the folder to a simple path like `C:\last30days-skill`
- right-click the app and choose Run as administrator
- make sure your internet connection works

If the app opens but does not return results:

- use a shorter topic
- try again in a minute
- check that the source sites are reachable
- restart the app and run the search again
